# 🎮 Stone Paper Scissors Game

A fun and interactive game built using **Python**. Play against the computer and see if you can win!

## 🚀 How to Play
1. Make sure you have Python installed.
2. Run the file using the command: `python game.py`
3. Enter your choice: `stone`, `paper`, or `scissors`.
4. The computer will randomly pick its move and the winner will be announced!

## 🛠️ Tech Used
- **Language:** Python
- **Modules:** `random`
